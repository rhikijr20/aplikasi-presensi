# 🚀 Aplikasi Presensi Laravel

Aplikasi presensi berbasis web yang dibangun menggunakan Laravel untuk mempermudah pencatatan kehadiran secara digital.
Project ini dibuat sebagai bagian dari pembelajaran dan pengembangan skill backend web development.

---

## ✨ Fitur Utama

* 🔐 Autentikasi (Login & Register)
* 📊 Dashboard Admin
* 📝 CRUD Data Presensi
* 👥 Manajemen User
* 📅 Pencatatan Kehadiran

---

## 🛠️ Tech Stack

* **Backend:** Laravel (PHP)
* **Frontend:** Blade, Bootstrap
* **Database:** MySQL
* **Tools:** Git & GitHub

---

## ⚙️ Cara Menjalankan Project

1. Clone repository

```bash
git clone https://github.com/rhikijr20/aplikasi-presensi.git
```

2. Masuk ke folder project

```bash
cd aplikasi-presensi
```

3. Install dependency

```bash
composer install
```

4. Copy file environment

```bash
cp .env.example .env
```

5. Generate application key

```bash
php artisan key:generate
```

6. Setup database di `.env`, lalu jalankan:

```bash
php artisan migrate
```

7. Jalankan server

```bash
php artisan serve
```

8. Buka di browser:

```
http://127.0.0.1:8000
```
